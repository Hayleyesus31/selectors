This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

class

class can be applied to several item on a page

Here's how we hook an class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:orange;

compound

compound selctor allow us to use more than one selector for a set of rules. the selector must be separeated by commas.

Here's how the CSS is applied to the element:

div#new-div,li,new-list, li.my-list{
    background-color:blue;
}

child

the child selector allows us to acess the elements that are immediatly inside the current elements.

Here's how the CSS is applied to the element:

ul.my-kids > li{
    background-color:green;
}

multiple classes

we can apply multuple classes by placing them in the class attribution with a space between

Here's how we hook an class attribute to an element:

class="add-border add-background"

Here's how the CSS is applied to the element:

    background-color:gold;
}
